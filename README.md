# Eggmaster Adventure Game

**Author:** Paresh Singh Kandel  
**Language:** Python (with HTML/CSS summary export)  
**Type:** Interactive Text-Based Adventurous Storytelling Game


## Overview
Eggmaster Adventure is a text-based Python game where players navigate a jungle by making choices that affect the outcomes. The game features branching storylines, random events, and an inventory/HP system. At the end of the adventure, a summary is exported as an HTML page showing the playerâ€™s journey, final health, and collected items.


## Features
- Decision-based gameplay with multiple outcomes  
- HP and inventory tracking  
- Randomized events affecting the story  
- Dynamic endings based on choices and player state  
- HTML summary generation for game recap (involving tracking of game states)  


## Installation
1.Clone the repository:
git clone https://github.com/pareshforeducation-ux/Eggmaster-Adventure-Game.git

2.Ensure Python 3.x is installed

3.Place all story text files (*.txt) in the same folder as gamecode.py

4.Include style.css for styled HTML output

## How to play
1.Read the instructions presented, read the storylines, and input your choices (1, 2, or 3)

2.Check your HP and inventory updates throughout the game

3.An HTML display will be presented upon; the HP dropping below 0 or reaching 0, losing the game due to choosing a bad decision, or completing the journey!
